# Building Constructor UK — Luxury Landing Page

A premium, high-performance Next.js 14 landing page for **Building Constructor UK**, crafted in a dark luxury architectural style.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis smooth scroll
- Lucide React icons

## Implemented Experience
- Full-screen **Blueprint Hero** with canvas line-drawing animation fading into luxury renovation imagery
- Sticky **glass navigation** with blur and gold border
- Reveal animations (fade + slide up) across all major sections
- Count-up stats on viewport entry
- Premium bento-style project showcase with hover zoom
- Sophisticated vertical process timeline with glowing gold line
- Minimalist private consultation form with gold-accented inputs
- Dark mode-first design language with generous whitespace and alignment

## Brand Styling
- Background: `#050505`
- Primary Text: `#FFFFFF`
- Accent: `#D4AF37`
- Heading font: Playfair Display
- Body font: Inter

## Run
```bash
npm install
npm run dev
```
